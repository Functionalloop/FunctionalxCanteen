# FunctionalCanteen

A modern, fast, and feature-rich campus dining application designed to streamline cafeteria operations. It provides an intuitive interface for both students and cafeteria staff, built with React, Vite, and Tailwind CSS.

## Features

- **Student Ordering:** Real-time digital menu, queue management, and order placement.
- **Campus Wallet:** Seamless digital payments and balance top-ups.
- **Vendor Dashboard:** Live demand queue, order management, and dynamic menu templating.
- **Real-Time Broadcasts:** Instant announcements sent directly from the vendor to students.

## Run Locally

**Prerequisites:** Node.js installed on your machine.

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure Environment:**
   Make sure you have your Firebase credentials correctly set up in the `.env` file based on `.env.example`.

3. **Start the development server:**
   ```bash
   npm run dev
   ```
